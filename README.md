# Transformer from Scratch using PyTorch

## Overview
This repository contains a complete implementation of the Transformer architecture from scratch using PyTorch, based on the paper "Attention Is All You Need". It is designed for neural machine translation (e.g., English to French or other language pairs).

## Features
- Full Transformer (Encoder + Decoder) implementation
- Multi-Head Attention
- Positional Encoding
- Training pipeline with early stopping and checkpointing
- Attention visualization notebook
- Configurable for different datasets and languages

## Setup

### Prerequisites
- Python 3.9+
- PyTorch

### Installation
```bash
git clone https://github.com/nareshAiNexus/transformer-from-scratch-pytorch.git
cd transformer-from-scratch-pytorch
pip install -r requirements.txt
```
